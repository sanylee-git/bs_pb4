💡 최근 주요 업데이트

  • v1.4 (2026-03-21) — 도움말 콘텐츠 대폭 확장 (사진 가이드, AI 원리, 오류 해결, FAQ, 이용약관, 캐릭터 도감 신규 추가).
  • v1.3 (2026-03-11) — SEO·GEO 최적화: JSON-LD 스키마, Key Takeaways, Author Bio, AdSense 단위 추가.
  • v1.2 (2026-03-05) — 분석 캐릭터를 7개에서 10개로 확장. 코베니·요시다·포치타 추가.

v1.4 — 2026년 03월 21일

콘텐츠 대폭 확장 & 신뢰성 강화 업데이트

  • NEW 사진 촬영 가이드 페이지 신규 추가 — 조명, 각도, 기기별 촬영 팁 완전 정복.
  • NEW AI 분석 원리 페이지 신규 추가 — Teachable Machine, TensorFlow.js, 클라이언트 사이드 AI 원리 상세 설명.
  • NEW 오류 해결 가이드 페이지 신규 추가 — 카메라 오류, 모델 로딩, 업로드 문제 증상별 해결책.
  • NEW 종합 FAQ 페이지 신규 추가 — 20개 이상의 자주 묻는 질문 답변.
  • NEW 이용약관 페이지 신규 추가 — 서비스 이용 조건, 면책 조항, 저작권 안내.
  • NEW 캐릭터 도감 페이지 신규 추가 — 10개 캐릭터 세계관·능력·성격 상세 소개.
  • IMPROVE 모든 페이지 footer에 확장 내비게이션 링크 추가.
  • IMPROVE sitemap.xml에 신규 페이지 7개 추가.

v1.3 — 2026년 03월 11일

SEO·GEO 최적화 & E-E-A-T 강화 업데이트

  • NEW 정적 JSON-LD 스키마 추가 (WebApplication, FAQPage, BreadcrumbList, Article).
  • NEW Key Takeaways 섹션 추가 (AI 크롤러·GEO 최적화용 구조화 요약).
  • NEW Author Bio 섹션 추가 (E-E-A-T 신뢰 지표 강화).
  • NEW Google AdSense 광고 단위 삽입.
  • NEW Google 사이트 인증 메타 태그 추가.
  • IMPROVE 각 페이지 article:published_time, article:modified_time 메타 태그 추가.
  • IMPROVE AI 리퍼러 감지 및 GEO 트래킹 강화 (ChatGPT, Perplexity, Gemini, Claude 등).

v1.2 — 2026년 03월 05일

캐릭터 확장 업데이트

  • 캐릭터 분석 캐릭터 7개 → 10개로 확장.
  • NEW 코베니 히가시야마 추가.
  • NEW 요시다 히로후미 추가.
  • NEW 포치타 추가.
  • IMPROVE AI 모델 재학습 — 10개 클래스 지원 버전으로 업데이트.
  • IMPROVE 캐릭터 가이드 카드 10개 캐릭터 전체 업데이트.
  • FIX 일부 모바일 기기에서 결과 카드 이미지 저장 시 레이아웃 오류 수정.

UI 개선 및 버그 수정 업데이트

  • NEW 결과 공유 기능 추가 — X(트위터)·Facebook·Threads 공유, PNG 이미지 저장.
  • NEW 플로팅 SNS 공유 버튼 추가.
  • NEW 별점 평가 기능 추가.
  • IMPROVE 모바일 반응형 레이아웃 개선.

  • FIX 파일 업로드 후 분석 버튼이 비활성화 상태로 유지되는 문제 수정.
