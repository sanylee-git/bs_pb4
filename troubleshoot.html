Troubleshooting

카메라·업로드 오류 해결 가이드

증상별 원인과 해결책을 단계적으로 안내합니다

💡 Key Takeaways

  • 카메라 안 켜짐 — 브라우저 권한 설정, 다른 앱 카메라 점유, HTTPS 환경 체크.
  • 모델 로딩 느림 — 최초 접속 시 5~15MB 다운로드 정상. 1분 이상이면 새로고침.
  • 업로드 실패 — JPG·PNG·WEBP만 지원. 파일 크기·손상 여부 확인.
  • 모바일 문제 — iOS Safari 12+, Android Chrome 80+ 권장. 브라우저 업데이트 확인.

📷 문제 1: 카메라가 켜지지 않아요

증상

카메라 탭에서 '분석 시작' 버튼을 눌렀을 때 카메라 화면이 나타나지 않거나, "카메라를 시작하세요" 안내 화면이 그대로 유지되는 경우.

원인 1: 브라우저 카메라 권한이 차단됨

해결 방법:

  • Chrome (PC/Android): 주소창 왼쪽 자물쇠 아이콘(🔒) 클릭 → 카메라 → "허용"으로 변경 → 페이지 새로고침
  • Safari (iOS): 설정 앱 → Safari → 카메라 → "묻기" 또는 "허용"으로 변경 → 사이트 재방문
  • Firefox: 주소창 왼쪽 차단 아이콘 클릭 → 카메라 권한 재설정 → 새로고침
  • Edge: 주소창 자물쇠 아이콘 → 권한 → 카메라 허용

원인 2: 다른 앱이 카메라를 점유 중

웹캠이나 카메라는 한 번에 하나의 앱/탭만 사용할 수 있습니다.

  • Zoom, Teams, Meet 등 화상회의 앱을 모두 종료하세요.
  • 다른 브라우저 탭에서 카메라를 쓰는 서비스가 없는지 확인하세요.
  • Windows: 작업 표시줄에서 카메라 앱이 실행 중이면 닫으세요.
  • 모든 관련 앱 종료 후 브라우저를 완전히 재시작하세요.

원인 3: HTTP 환경에서 접속 (보안 오류)

브라우저 보안 정책상 카메라 접근은 HTTPS 또는 localhost에서만 허용됩니다. 주소창이 https://로 시작하는지 확인하세요. http://인 경우 카메라를 사용할 수 없습니다.

원인 4: 오래된 브라우저

카메라 API(WebRTC)는 최신 브라우저에서만 안정적으로 동작합니다. Chrome 80+, Safari 12+, Firefox 75+, Edge 80+ 이상으로 업데이트하세요.

⏳ 문제 2: 모델 로딩이 너무 오래 걸려요

증상

"모델을 준비 중입니다..." 또는 "모델 불러오는 중..." 메시지가 1분 이상 지속됩니다.

정상 로딩 시간

  • 빠른 인터넷 (50Mbps+): 3~10초
  • 일반 인터넷 (10Mbps): 10~25초
  • 느린 인터넷/모바일 데이터: 30~60초
  • 두 번째 방문부터는 브라우저 캐시로 인해 더 빠릅니다.

해결 방법

  • 인터넷 연결 상태를 확인하세요. Wi-Fi가 불안정하다면 재연결하세요.
  • 1분 이상 대기 후에도 변화가 없으면 페이지를 새로고침(F5 또는 Ctrl+R)하세요.
  • 브라우저 캐시를 지우고 재시도: Chrome은 Ctrl+Shift+Delete → 캐시된 이미지 및 파일 삭제.
  • VPN을 사용 중이라면 잠시 해제 후 시도해보세요. 일부 VPN이 CDN 연결을 방해할 수 있습니다.
  • 회사·학교 네트워크의 방화벽이 jsdelivr.net 또는 teachablemachine.withgoogle.com을 차단하는 경우 Wi-Fi가 아닌 모바일 데이터로 시도해보세요.

🖼️ 문제 3: 이미지 업로드가 안 돼요

증상

파일을 선택해도 미리보기가 뜨지 않거나, 분석 시작 버튼이 활성화되지 않는 경우.

해결 방법

  • 파일 형식 확인: JPG(.jpg, .jpeg), PNG(.png), WEBP(.webp) 형식만 지원합니다. HEIC(아이폰 기본 포맷)나 BMP, TIFF 등은 지원하지 않습니다.
  • HEIC 변환 (iPhone): 아이폰 설정 → 카메라 → 포맷 → "호환성 우선"으로 변경하면 이후 사진은 JPG로 저장됩니다. 또는 온라인 변환 도구로 HEIC → JPG 변환 후 업로드하세요.
  • 파일 크기: 매우 큰 파일(50MB 이상)은 처리 시간이 늘어날 수 있습니다. 일반적인 스마트폰 사진(2~10MB)은 문제없습니다.
  • 파일 손상: 파일 탐색기에서 해당 사진이 정상적으로 열리는지 먼저 확인하세요.

모바일 이미지 업로드 방법

  • 업로드 영역을 탭하면 갤러리 또는 카메라 선택 메뉴가 나타납니다.
  • iOS에서 처음 업로드 시 사진 접근 권한 요청이 나타납니다. "허용"을 선택하세요.
  • Android에서 파일 피커가 열리지 않는 경우 브라우저 앱 권한(저장소/미디어)을 설정에서 확인하세요.

📱 문제 4: 모바일에서 화면이 이상하게 보여요

증상

레이아웃이 깨지거나, 버튼이 너무 작거나, 텍스트가 겹쳐 보이는 경우.

해결 방법

  • 브라우저 줌 초기화: 브라우저 설정에서 페이지 확대/축소를 100%로 초기화하세요.
  • 브라우저 업데이트: App Store 또는 Google Play에서 최신 버전으로 업데이트하세요.
  • 가로모드 시도: 스마트폰을 가로로 돌리면 AI 분석 카드 레이아웃이 더 잘 보이는 경우가 있습니다.
  • Safari Private 모드 제한: iOS Safari의 프라이빗 탭에서는 일부 기능이 제한될 수 있습니다. 일반 탭에서 사용해보세요.

Samsung 인터넷 브라우저

삼성 인터넷 브라우저도 지원하지만, 카메라 권한 처리 방식이 Chrome과 약간 다를 수 있습니다. 문제가 발생하면 Chrome 앱을 설치해 사용해보세요.

🔍 문제 5: 분석 결과가 항상 같은 캐릭터만 나와요

증상

여러 번 시도해도 특정 캐릭터가 계속 나오거나, 유사도가 100%에 가까운 극단적인 결과만 나옵니다.

원인과 해결

  • 조명 편향: 특정 색온도나 밝기의 조명이 특정 캐릭터의 학습 데이터와 유사할 수 있습니다. 다른 조명 환경에서 시도해보세요.
  • 배경 영향: 배경의 색상이나 패턴이 분류에 영향을 줄 수 있습니다. 단순한 배경 앞에서 시도해보세요.
  • 표정 변화: 웃는 표정, 무표정, 놀란 표정 등 다양하게 시도해보세요.
  • 사진 교체: 다른 날, 다른 장소에서 찍은 사진으로 시도해보면 다른 결과를 얻을 수 있습니다.

이는 오류가 아닌 정상 동작입니다. 엔터테인먼트 서비스이므로 다양하게 시도해보며 재미있게 즐기시기 바랍니다.
